# Hello React Front end

This is Microverse project for learning how to build React application, consuming data from Rails Api , using Redux for menaging state and dynamic routing.

## Built With

- ReactJS
- Redux and react-redux

## Installation

Go to your terminal, navigate to your working directory and run

`git clone https://github.com/Nuri1977/hello-react-front-end.git`

After that navigate to the newly created folder

`cd hello-react-front-end`

You now have access to the files.
To install necessary dependencies run

`npm install`

Start a local server running the command

`npm start`

You should now have the project running locally on a dev server.

`npm test`

Launches the test runner in the interactive watch mode.
